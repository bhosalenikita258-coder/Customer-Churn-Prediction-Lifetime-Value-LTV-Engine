# Customer Churn Prediction & Customer Lifetime Value Analysis

## Project Overview
This project aims to analyze customer behavior and predict customer churn using machine learning techniques. It helps identify customers who are likely to leave and provides insights for improving customer retention.

## Objectives
- Analyze customer demographics and service usage.
- Identify factors affecting customer churn.
- Perform data cleaning and preprocessing.
- Build a machine learning model for churn prediction.
- Generate business insights from customer data.

## Dataset
The dataset contains customer information such as:
- Demographic details
- Service subscriptions
- Contract information
- Monthly and total charges
- Churn status

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Project Workflow

### 1. Data Loading and Understanding
- Imported required libraries.
- Loaded the customer churn dataset.
- Explored dataset structure and features.

### 2. Data Cleaning and Preprocessing
- Handled missing values.
- Converted data types where required.
- Removed unnecessary columns.
- Encoded categorical variables.

### 3. Exploratory Data Analysis (EDA)
- Analyzed customer churn distribution.
- Studied the impact of contract types on churn.
- Examined monthly charges and tenure patterns.
- Performed correlation analysis.

### 4. Feature Engineering
- Created new features to improve model performance.
- Prepared data for machine learning.

### 5. Model Building and Evaluation
- Split data into training and testing sets.
- Applied feature scaling.
- Trained a Logistic Regression model.
- Evaluated performance using accuracy and classification metrics.

## Results
The analysis identified important factors influencing customer churn and developed a predictive model to help businesses improve customer retention strategies.
